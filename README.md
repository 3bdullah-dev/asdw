# بورتفوليو مونتير فيديوهات احترافي

موقع ويب بورتفوليو احترافي لمونتير فيديوهات مع تصميم حديث ومستجيب لجميع الأجهزة.

## المميزات

- ✅ تصميم احترافي وحديث مناسب لمونتير فيديوهات
- ✅ مستجيب بالكامل لجميع الأجهزة (موبايل، تابلت، ديسكتوب)
- ✅ قسم رئيسية مع صورة شخصية وشعار
- ✅ قسم "من أنا" مع وصف وقائمة العملاء
- ✅ قسم "أعمالي" لعرض فيديوهات من YouTube مع مميزات المشروع وأيقونات البرامج
- ✅ قسم "تواصل معي" مع نموذج تواصل
- ✅ تنقل سلس بين الأقسام
- ✅ تأثيرات بصرية احترافية

## الملفات

- `index.html` - الملف الرئيسي للصفحة
- `style.css` - ملف التصميم والأنماط
- `script.js` - ملف JavaScript للتفاعلية
- `config.js` - ملف الإعدادات لتخصيص المحتوى

## كيفية الاستخدام

### 1. تخصيص المحتوى

افتح ملف `config.js` وعدّل المحتوى حسب احتياجاتك:

#### تحديث المعلومات الشخصية:
```javascript
profile: {
    imageUrl: "رابط صورتك الشخصية",
    tagline: "شعارك المهني"
}
```

#### تحديث قسم "من أنا":
```javascript
about: {
    description: "وصفك الشخصي والمهني",
    clients: ["العميل 1", "العميل 2", ...]
}
```

#### إضافة مشاريعك:
```javascript
portfolio: [
    {
        title: "اسم المشروع",
        description: "وصف المشروع",
        videoId: "VIDEO_ID", // من رابط YouTube
        features: ["ميزة 1", "ميزة 2"],
        software: ["premier", "after-effects", "capcut"]
    }
]
```

### 2. الحصول على Video ID من YouTube

من رابط YouTube مثل:
```
https://www.youtube.com/watch?v=VIDEO_ID
```

استخدم `VIDEO_ID` فقط في حقل `videoId`.

### 3. رفع الصور

- ضع صورتك الشخصية في مجلد المشروع أو استخدم رابط مباشر
- أو استخدم خدمة استضافة صور مثل Imgur

### 4. فتح الموقع

افتح ملف `index.html` في المتصفح أو استخدم خادم محلي:

```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx http-server
```

ثم افتح المتصفح على `http://localhost:8000`

## تخصيص الألوان

يمكنك تعديل الألوان من ملف `style.css` في قسم `:root`:

```css
:root {
    --primary-color: #ff006e;
    --secondary-color: #8338ec;
    --accent-color: #3a86ff;
    /* ... */
}
```

## البرامج المدعومة

- Adobe Premier Pro
- After Effects
- CapCut Pro

## المتطلبات

- متصفح حديث (Chrome, Firefox, Safari, Edge)
- اتصال بالإنترنت (لتحميل Font Awesome)

## الدعم

لأي استفسارات أو مشاكل، يمكنك التواصل من خلال قسم "تواصل معي" في الموقع.
